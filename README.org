#+TITLE: Gritcode
#+SUBTITLE: Native coding engine

[[https://gritcode.ai][gritcode.ai]]

GPU-rendered AI coding harness built in C++ with OpenGL, HarfBuzz, and FreeType.

⚠️ *Warning:* Gritcode does not have a permission system. AI agents can execute any command on your machine. Press *Escape* to cancel a running request at any time.

** Features

- *GPU-rendered UI* -- OpenGL 3.3 batched quads, SDF rounded rects, 0% idle CPU
- *HarfBuzz text shaping* -- proper Arabic RTL, ligatures, contextual forms, diacritics
- *Emoji support* -- FontConfig fallback, ZWJ sequences, bitmap scaling
- *Multi-provider* -- [[https://opencode.ai/zen][OpenCode]] Zen (pay-as-you-go), OpenCode Go (quality coding models, low monthly price), and Claude ACP (requires [[https://docs.anthropic.com/en/docs/claude-code/overview][Claude Code CLI]])
- *Tool use* -- bash, read_file, list_directory with background execution
- *Session persistence* -- per working directory, automatic restore
- *Markdown rendering* -- bold, italic, code blocks, headings, blockquotes, lists
- *~9,400 lines of C++* -- entire codebase

** Is this production ready?

No. This is not even pre-alpha. I use Gritcode every day as my primary coding environment, but every few hours I hit a bug — sometimes small, sometimes not. Things break, things get fixed, things change fast. This is a live project under active development.

If you want to see it stabilize faster, [[https://github.com/sponsors/lszl84][sponsor the project]] 💜

** Quick Start

*** Linux (Arch)

#+begin_src bash
sudo pacman -S glfw freetype2 harfbuzz fontconfig curl libsecret

git clone https://github.com/lszl84/gritcode.git
cd gritcode
cmake -B build -G Ninja -DCMAKE_BUILD_TYPE=Release -DCMAKE_INSTALL_PREFIX=~/.local
cmake --build build
cmake --install build

grit
#+end_src

*** Linux (Ubuntu/Debian)

#+begin_src bash
sudo apt install libglfw3-dev libfreetype-dev libharfbuzz-dev libfontconfig-dev libcurl4-openssl-dev libsecret-1-dev cmake ninja-build

git clone https://github.com/lszl84/gritcode.git
cd gritcode
cmake -B build -G Ninja -DCMAKE_BUILD_TYPE=Release -DCMAKE_INSTALL_PREFIX=~/.local
cmake --build build
cmake --install build

grit
#+end_src

*** macOS

#+begin_src bash
brew install cmake ninja

git clone https://github.com/lszl84/gritcode.git
cd gritcode
cmake -B build -G Ninja -DCMAKE_BUILD_TYPE=Release
cmake --build build

open build/grit.app
#+end_src

On macOS, GLFW, FreeType, HarfBuzz, and curl are fetched and built from source automatically by CMake.

** Architecture

| File              | Purpose                                     |
|-------------------+---------------------------------------------|
| =app.cpp=         | Main app, providers, streaming, tool use    |
| =gl_renderer.cpp= | OpenGL batched renderer, SDF shapes         |
| =font.cpp=        | HarfBuzz shaping, FreeType raster, fallback |
| =scroll_view.cpp= | Block layout, word wrap, selection, caret   |
| =markdown_renderer.cpp= | cmark AST to styled blocks             |
| =widgets.cpp=     | TextInput, ComboBox, Button (custom)        |
| =glfw_window.cpp= | GLFW setup, input handling, main loop       |
| =curl_http.cpp=   | Async HTTP with SSE streaming               |
| =session.cpp=     | Per-CWD session persistence                 |
| =keychain.cpp=    | libsecret / Keychain API key storage        |

** Requirements

- Linux (Wayland or X11) or macOS
- CMake 3.24+, Ninja, C++23 compiler
- Linux: glfw3, freetype2, harfbuzz, fontconfig, libcurl, libsecret-1, OpenGL
- macOS: Xcode command line tools (dependencies fetched automatically)

** Contributing

✨ *This is all vibe coded.* ✨

Not a single line of code in this project was typed by hand. Every feature, every fix, every refactor — all driven through AI agents. That's the whole point.

I don't intend to write code manually, and as the project grows the focus will be on building features for more surgical agentic interventions — interaction graphs for code components, visual code browsing, targeted refactoring, debugging, performance profiling. The IDE itself is built the way it's meant to be used.

*Pull requests are not accepted for now.* If you find a bug or have a suggestion, please open an issue — and if you can, run your own AI agent to analyze the problem and include its suggestions in the report. That kind of contribution is worth more than a patch. Thanks! 🙏
